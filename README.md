# Beta Tech Solutions Web-Based System

## 📌 Project Overview
This project is a collaborative web-based system developed using HTML as part of a group assignment at Beta Tech Solutions Ltd.

The system demonstrates the use of Git and GitHub for version control and teamwork in a remote development environment.

---

## 🎯 Objectives
- To design and implement multiple web forms using HTML
- To practice collaborative development using GitHub
- To understand branching, commits, and pull requests
- To integrate individual work into a single system

---

## 🧩 Features
The system includes the following forms:

- 🔐 User Login Form
- 👨‍💼 Employee Registration Form
- 🛒 Product Registration Form
- 💬 Customer Feedback Form
- 📅 Event Booking Form

Each form is developed by a different team member and integrated into the main project.

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Git
- GitHub

---

## 🔀 GitHub Workflow
- A shared repository is used for the project
- Each developer works on a separate branch
- Changes are committed with meaningful messages
- Pull Requests are created for review before merging
- All features are merged into the main branch after approval
